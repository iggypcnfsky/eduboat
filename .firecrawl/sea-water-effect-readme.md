# Sea Water Effect (Three.js)

[Permalink: Sea Water Effect (Three.js)](https://github.com/ramesh-852000/sea-water-effect-three-js#sea-water-effect-threejs)

A 3D sea water effect created using **Three.js** and **WebGL shaders**.

This project demonstrates how to build realistic ocean/sea animations with waves, reflections, and dynamic lighting.

## 🌊 Features

[Permalink: 🌊 Features](https://github.com/ramesh-852000/sea-water-effect-three-js#-features)

- Realistic animated sea surface
- Wave simulation using shaders
- Reflection and refraction effects
- Responsive rendering with Three.js

## 🛠️ Tech Stack

[Permalink: 🛠️ Tech Stack](https://github.com/ramesh-852000/sea-water-effect-three-js#%EF%B8%8F-tech-stack)

- [Three.js](https://threejs.org/)
- WebGL / GLSL Shaders
- JavaScript (ES6)
